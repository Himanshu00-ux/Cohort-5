# Digital Voting System

##  Overview
The Digital Voting System is an embedded systems project that simulates an electronic voting process using a microcontroller. The system allows users to cast votes through push buttons and automatically displays the final voting results after the voting session ends.

This project demonstrates basic concepts of digital electronics, microcontroller programming, and automated vote counting.

---

##  Objective
- To design a simple and reliable electronic voting system  
- To eliminate manual vote counting errors  
- To display voting results automatically  

---

##  SDG Alignment
**SDG 16 – Peace, Justice and Strong Institutions**  
The project promotes transparency and fairness through digital voting automation.

---

##  Features
- Multiple candidates using 3–4 voting buttons  
- Real-time vote counting  
- Dedicated button to end voting  
- Automatic result display  
- Simple and user-friendly operation   

---

##  Working Principle
1. Each push button represents a candidate.
2. When a voter presses a button, the vote count increases.
3. Votes are stored during the voting process.
4. Pressing the End Voting button stops further voting.
5. Total votes received by each candidate are displayed.

---

##  Applications
- Educational demonstrations  
- Classroom or small-scale elections  
- Embedded systems learning projects
